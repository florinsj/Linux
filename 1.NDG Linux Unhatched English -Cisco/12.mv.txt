	mv people.csv Work				// move file people.csv in Word directory
	mv numbers.txt letters.txt alpha.txt School	// move specified files to School directory

	mv animals.txt zoo.txt		// animals.txt to zoo.txt  (move animals.txt to same directory but with another name)

Permissions:

	mv needs write and eXecute to both destination directories
